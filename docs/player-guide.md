# Earth Online 玩家入门流程

这个 mod 面向“不需要先学化学也能玩”的玩家。核心玩法是：挖到更真实的矿床和岩石，用机器一步步处理，最后仍然得到 MC 和其他 mod 熟悉的锭、宝石、红石、青金石等物品。

## 第一步：拿到手册

合成 `野外地质手册`：

```text
任意一块泥土
或任意木板
或任意石头
=> 野外地质手册
```

右键手册会打开分页界面，左侧切换章节，右侧查看路线说明。这个配方故意很便宜，因为它是新玩家的说明书，不应该变成解谜门槛。

空手右键任意处理机器会打开该机器 GUI。手持可处理材料右键机器，会直接处理 1 个材料并给出多种产物。机器界面里也有 `打开手册` 按钮，测试路线时不用退出界面重新找书。

如果不知道某个材料下一步去哪，直接把鼠标停在 Earth Online 物品上。tooltip 会显示：

```text
下一步：应该放入哪台机器
示例产出：第一条可用路线会得到什么
常见来源：这个材料通常从哪里来
```

装了 JEI 时，可以在 `Earth Online 工业处理` 分类里查全部路线。

## 本体和附属

本体当前只覆盖现实地球基础：地质、矿床、岩石、土壤、水文、现代工业、电力、基础设施、手册和 JEI 路线。

这些内容先不放进核心：

- 地球人附属：真实人物属性、体温、营养、疾病、食物保存和医疗系统。
- 修仙附属：灵气、灵根、丹药、法宝、洞天和境界。
- 魔法附属：魔力、元素、仪式、符文、炼金和魔法生态。
- 科幻附属：太空、高级能源、AI、纳米和超现实自动化。

这样做是为了让核心 jar 适合所有玩家和整合包；附属可以改变玩法压力或世界观，但必须可选。

## 基础流程

```text
矿石/矿床
-> 颚式破碎机
-> 碎块
-> 球磨机
-> 粉末
-> 筛分 / 磁选 / 浮选 / 浸出
-> 精矿
-> 焙烧 / 还原 / 电解 / 精馏 / 合成 / 聚合
-> 原版锭、宝石、红石等兼容产物
```

## 典型路线

快速索引：

```text
矿床/矿石方块 -> 颚式破碎机
矿物碎块 -> 球磨机
磁铁矿粉/赤铁矿粉 -> 磁选机
黄铜矿粉/金粉/红石矿物 -> 浮选槽
精矿 -> 焙烧炉 / 还原炉 / 电解槽
盐粉 -> 电解槽，进入氯碱工业
煤粉/煤焦油/煤气 -> 气体分离器 / 精馏塔 / 蒸汽裂解炉
硫粉/氨/甲醇/乙烯/氯化钾 -> 化学反应釜或合成塔
长石粉/云母粉/铝硅酸盐粉 -> 筛分机 / 工业窑炉
水泥粉/塑料树脂 -> 压粉机
```

铁：

```text
磁铁矿矿石 Fe3O4
-> 破碎机：磁铁矿碎块 + 尾粉
-> 球磨机：磁铁矿粉
-> 磁选机：铁精矿 + 尾粉
-> 还原炉：minecraft:iron_ingot + 矿渣
```

铜：

```text
黄铜矿矿石 CuFeS2
-> 破碎机：黄铜矿碎块 + 黄铁矿粉
-> 球磨机：黄铜矿粉
-> 浮选槽：铜精矿 + 黄铁矿粉 + 尾粉
-> 焙烧炉：焙烧铜精矿 + 硫粉
-> 还原炉/电解槽：minecraft:copper_ingot + 副产物
```

金：

```text
含金石英脉
-> 破碎机：含金石英碎块 + 石英粉
-> 筛分机/浸出槽：金粉或金精矿 + 二氧化硅粉 + 尾粉
-> 还原炉/电解槽：minecraft:gold_ingot
```

钻石：

```text
含金刚石金伯利岩
-> 破碎机：金伯利岩碎块 + 金刚石砂粒
-> 筛分机：更多金刚石砂粒
-> 压粉机：minecraft:diamond
```

## 岩石不是单一化学式

原版花岗岩、闪长岩、安山岩、深板岩、凝灰岩、方解石、滴水石、砂岩、黑石、玄武岩等都被视为混合岩石。它们不会假装只有一个化学式，而是通过球磨产出多种粉末。

例子：

```text
花岗岩
-> 球磨机
-> 石英粉 + 长石粉 + 云母粉 + 尾粉
```

现实世界不能简单磨一块石头就得到 100% 纯物质；Earth Online 的折中是：处理路线尽量真实，但最终允许 MC 世界用机器/合成魔法整理成可堆叠、可合成的物品。

## 常见化学工业

氯碱：

```text
盐粉 NaCl
-> 电解槽
-> 氢氧化钠 + 氯气单元 + 氢气单元
```

合成氨 / 尿素 / 肥料：

```text
玻璃瓶
-> 气体分离器：氮气单元 + 氧气单元
-> 合成塔：氨
-> 合成塔：尿素
-> 混合机 / 肥料造粒机：复合肥颗粒
```

煤化工 / 塑料：

```text
煤粉
-> 气体分离器：焦炭 + 煤焦油 + 煤气单元
-> 蒸汽裂解炉：苯 + 乙烯 + 丙烯
-> 聚合釜：聚乙烯 / 聚丙烯 / PVC 树脂
```

石油炼化 / 现代塑料：

```text
烟煤含煤岩
-> 颚式破碎机：煤粉 + 煤焦油 + 煤气单元 + 原油样品 + 尾粉
-> 精馏塔：天然气 + 石脑油 + 煤油 + 柴油 + 润滑油 + 沥青 + 石油焦
-> 蒸汽裂解炉：乙烯 + 丙烯 + 苯 + 炭黑
-> 反应釜 / 合成塔 / 聚合釜：PS、PET、合成橡胶、尼龙纤维
```

无烟煤含煤岩更偏高碳路线：破碎后给更多煤粉、石墨粉和尾粉，更适合石墨、电池碳、活性炭和高热值燃料链。

如果只想快速测试：

```text
原油样品 -> 精馏塔
石脑油 -> 蒸汽裂解炉
苯 -> 反应釜 -> 苯乙烯 / 己内酰胺
苯乙烯 -> 聚合釜 -> 聚苯乙烯树脂
己内酰胺 -> 聚合釜 -> 尼龙纤维
```

造纸 / 纤维：

```text
原木
-> 球磨机：木片
-> 反应釜：纤维素浆
-> 吸收塔：漂白浆
-> 压粉机：纸
```

颜料 / 涂料：

```text
砂 -> 筛分机：二氧化钛
赤铁矿粉 -> 反应釜：氧化铁颜料
石脑油/柴油/润滑油路线 -> 炭黑
二氧化钛 / 氧化铁颜料 / 炭黑 -> 混合机：涂料基料 + 染料
```

电池材料：

```text

炭黑
-> 反应釜：活性炭 + 电池级碳粉

盐卤结晶

红石精矿

石墨粉 / 锰氧化物粉 / 镍前驱体
-> 混合机：电极片

锂盐
-> 反应釜：电解液

电解液 / 电极片
```
