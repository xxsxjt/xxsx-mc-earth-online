# Earth Online 第二轮工业扩展计划

## 本轮目标

第二轮不追求一次性复刻全部现代工业，而是把第一版从“矿物处理 + 基础化工可玩原型”推进到“近现代工业生态雏形”。

核心标准：

- 玩家拿到新材料后，tooltip、JEI 和手册都能告诉他下一步。
- 每条路线至少有一个清晰的自然来源、一个工业过程、一个 MC 兼容出口。
- 新内容优先复用现有机器，只有玩法确实需要时才新增机器。
- 先把路线做成可测、可玩、可讲明白，再升级多输入、流体、电力和污染系统。

## 优先级 A：第二轮必须先做

### 1. 电池与电化学材料

理由：电池工业是现代工业辨识度很高的入口，也能自然连接石墨、盐、铝、铜、镍/锰/铁等矿物系统。

第一批物品：

- 石墨粉
- 活性炭
- 电池级碳粉
- 锰氧化物粉
- 镍前驱体
- 锂盐
- 电解液
- 电极片
- 简易电池单元

路线草案：

```text
煤粉 / 石油焦 -> 焙烧炉 / 反应釜 -> 活性炭 + 石墨粉
红石矿物 / 黑石粉系 -> 浮选 / 磁选 -> 锰氧化物粉
盐卤结晶 -> 结晶器 -> 锂盐 + 氯化钾 + 盐粉
石墨粉 + 金属前驱体 -> 混合机 -> 电极片
电极片 + 电解液 -> 电解槽 / 合成塔 -> 简易电池单元
简易电池单元 -> 压粉机 -> 红石 / 铜锭 / 兼容科技材料
```

第一版折中：电池暂时输出红石、铜锭或未来兼容标签物品，不先实现能量系统。

### 2. 水处理与环保闭环

理由：当前尾粉、矿渣、副产酸碱已经出现，水处理能让“工业副产物不是垃圾”这件事更完整。

第一批物品：

- 硬水样品
- 软化水
- 污泥饼
- 中和盐
- 活性炭滤料
- 石灰处理渣
- 稳定化尾矿

路线草案：

```text
水桶 -> 混合机 / 反应釜 -> 硬水样品
硬水样品 + 石灰粉路线 -> 软化水 + 石灰处理渣
酸类 / 烧碱 / 熟石灰 -> 反应釜 -> 中和盐
尾粉 + 水处理材料 -> 混合机 -> 稳定化尾矿
活性炭 -> 吸收塔 -> 活性炭滤料
稳定化尾矿 -> 压粉机 -> 石头 / 建材兼容物
```

第一版折中：仍然用单输入路线表达“处理方向”，未来多输入系统再加入真实中和配比。

### 3. 橡胶与弹性材料

理由：合成橡胶已经有入口，但还不够像完整工业；橡胶也能连接炭黑、硫粉、石油和 MC 黏液球。

第一批物品：

- 天然胶乳
- 未硫化橡胶
- 硫化橡胶
- 橡胶复合料
- 密封圈

路线草案：

```text
藤蔓 / 树叶 / 木片 -> 反应釜 -> 天然胶乳
天然胶乳 / 合成橡胶 -> 混合机 -> 未硫化橡胶
未硫化橡胶 + 硫粉路线 -> 工业窑炉 / 反应釜 -> 硫化橡胶
硫化橡胶 + 炭黑路线 -> 混合机 -> 橡胶复合料
橡胶复合料 -> 压粉机 -> 黏液球 / 密封圈
```

第一版折中：密封圈先作为 Earth Online 中间物，用于未来机器升级，不急着绑定其他 mod。

### 4. 基础有机溶剂与树脂

理由：塑料路线已经有乙烯、丙烯、苯、甲醇、甲醛；第二轮应补上乙醇、乙酸、丙酮、酚醛树脂、环氧树脂入口。

第一批物品：

- 乙醇
- 乙酸
- 丙酮
- 苯酚
- 酚醛树脂
- 环氧树脂
- 工业溶剂

路线草案：

```text
糖 / 小麦 / 木片 -> 反应釜 -> 乙醇
乙醇 -> 反应釜 -> 乙酸
丙烯 -> 反应釜 -> 丙酮
苯 -> 反应釜 -> 苯酚
苯酚 / 甲醛 -> 合成塔 -> 酚醛树脂
丙酮 / 苯酚路线 -> 合成塔 -> 环氧树脂
工业溶剂 -> 吸收塔 / 反应釜 -> 回收副产物
```

第一版折中：不表现毒性和易燃风险，只用手册文字说明现实注意事项。

## 优先级 B：第二轮后半段做

### 5. 现代冶金升级

目标：让钢、铝、硅铁不再只是孤立中间物。

候选内容：

- 不锈钢坯
- 锰铁
- 铬铁
- 铝合金坯
- 镁粉
- 钛白副产钛渣

路线方向：

```text
铁锭 / 钢坯 -> 合金混合 -> 不锈钢坯
铝锭 + 镁粉 / 硅铁 -> 合金坯
钛白粉路线副产 -> 钛渣 -> 后续钛工业入口
```

出口：铁锭、铁块、盾牌/桶/矿车等兼容物，未来再接机器外壳升级。

### 6. 半导体前置材料

目标：先做“高纯硅和基础电子材料”，不要一上来做完整芯片工业。

候选内容：

- 冶金级硅
- 高纯硅粉
- 氯硅烷
- 多晶硅
- 硅晶圆
- 掺杂剂粉
- 光刻胶前驱体

路线方向：

```text
二氧化硅粉 -> 还原炉 -> 硅铁 / 冶金级硅
冶金级硅 -> 反应釜 -> 氯硅烷
氯硅烷 -> 精馏塔 -> 高纯硅粉
高纯硅粉 -> 压粉机 -> 硅晶圆
苯 / 丙酮 / 树脂路线 -> 光刻胶前驱体
```

出口：红石、玻璃、石英、未来科技联动物品。

### 7. 稀土与磁材

目标：为后续电机、磁选机升级、电池和高科技材料做储备。

候选内容：

- 独居石砂
- 氟碳铈矿粉
- 混合稀土氧化物
- 钕盐
- 钕铁硼磁材
- 稀土尾渣

路线方向：

```text
砂 / 花岗岩 / 黑石筛分 -> 稀土矿砂
稀土矿砂 -> 浸出槽 -> 混合稀土氧化物 + 稀土尾渣
混合稀土氧化物 -> 结晶器 -> 钕盐
钕盐 + 铁路线 -> 合成塔 -> 钕铁硼磁材
```

出口：磁选机升级材料、红石、指南针等兼容物。

## 优先级 C：系统升级，不宜和内容大包混在一起

### 1. 数据驱动路线

当前处理路线硬编码在 `ProcessingMachineBlock.createRecipes()`，好处是稳定、易调试；坏处是内容越来越多后难维护。

第二轮后建议拆出数据结构：

```text
data/earth_online/processing_routes/*.json
```

目标字段：

- machine
- input item/tag
- output item/count 列表
- note
- handbook category
- optional compatibility namespace

先做读取与展示，不急着让全部路线立刻迁移。

### 2. 多输入机器与流体

现代化工很多流程本质上需要多输入和流体。建议等路线数量稳定后再做：

- BlockEntity 储存输入/输出
- Menu + Screen 显示槽位
- 处理时间
- 流体或“桶/单元”简化流体
- 能耗接口预留

第一台适合升级的机器：化学反应釜。

原因：酸碱中和、橡胶硫化、电池电解液、树脂合成都需要多输入。

### 3. 手册升级

当前手册是静态分页，适合第一版。第二轮内容增加后建议做：

- 搜索框
- 按机器筛选
- 按产业链筛选
- “我手里有这个”反查
- 点击条目跳 JEI 配方
- 区分原版核心路线和未来 mod 联动路线

## 下一次连续实现建议

下一次不要同时做 A/B/C 全部内容。建议只做 A 组前两块：

1. 电池与电化学材料
2. 水处理与环保闭环

原因：

- 能复用煤、盐、红石、石灰、尾粉、矿渣等既有内容。
- 玩家能立刻看到工业链从“挖矿冶炼”扩展到“副产物治理和现代材料”。
- 不需要先改机器架构，仍可用现有单输入路线跑通 JEI、tooltip 和手册。
- 内容量适中，适合一次构建、一次部署、一次进游戏测试。

下一次实现批次预估：

- 新增 14 到 18 个物品。
- 新增 25 到 35 条处理路线。
- 新增 2 到 3 页手册。
- 更新 Agnes 贴图脚本提示词。
- 构建、资源校验、部署、提交推送。

## 暂缓项

这些值得做，但不建议塞进下一次连续实现：

- 完整石油油田/天然气田世界生成。
- 多输入化工反应釜。
- 真实流体系统。
- 污染、毒性、爆炸和安全事故。
- 其他大型科技 mod 联动物品。
- 完整半导体芯片产业。

原因很简单：这些会把测试复杂度从“路线验证”直接推到“系统验证”，很容易把第一版打磨节奏拖死。
