# Earth on Minecraft 科幻附属路线图

## 定位

科幻附属不是 Earth on Minecraft 本体的一部分。它应依赖本体的现实材料、电网、核热、半导体、稀土磁材、PCB、自动化总线和机器人部件，再向“现实工程之外”的内容扩展。

本体继续负责真实世界常见内容：矿物、冶金、化工、核能、电网、PLC、传感器、输送、机器人、基础设施和现代工业。科幻附属只做太空、AI 自治社会、纳米制造、反物质、曲率、超导/等离子高级能源、星际材料等超出现实工程常识的路线。

## 启动条件

先不抢本体开发节奏。至少满足这些条件后再开科幻附属模块：

- 本体核热 -> 汽轮发电机 -> 电缆 -> 电池箱 -> 机器耗电闭环稳定。
- 本体输送机、分拣器、机器输入/输出接口形成基本物流闭环。
- 本体手册、JEI、tooltip 能让新手知道材料去向。
- 本体贴图和多方块结构达到“能辨认真实设备”的最低审美线。

## 第一批科幻内容

第一版科幻附属应从“高级用途”开始，不反向修改本体开局难度：

- 高级计算：高纯硅、晶圆、光刻胶、PCB、稀土磁材 -> 量子/光子计算部件。
- 智能工厂：PLC、自动化总线、机器人臂、机器视觉 -> 自主工厂控制核心。
- 高级能源：核热模块、储能、电网开关柜、超导材料 -> 聚变/等离子/高密度储能前置。
- 太空结构：钛、铝合金、陶瓷、碳材料、聚合物 -> 真空结构、隔热瓦、空间站框架。
- 纳米制造：半导体、催化剂、陶瓷、稀土 -> 纳米加工平台和高端材料升级。

## 不做

- 不做枪炮武器链。
- 不把真实人体难度塞进科幻附属；地球人类是独立联动模组。
- 不在科幻附属里重写本体矿物/化工主线；只读取本体产物和标签。
