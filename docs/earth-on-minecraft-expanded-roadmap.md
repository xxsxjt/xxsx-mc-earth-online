# Earth on Minecraft 扩展路线图

## 定位

Earth on Minecraft 是现实世界内容复刻模组，不是单线科技模组，也不是硬核生存难度模组。

核心规则：

- 复刻内容，不复刻痛苦。保留 MC 的合成魔法、背包堆叠、原版锭/宝石/红石生态和整合包兼容出口。
- 优先做现实中常见、能被玩家理解的路线：地质、矿产、冶金、化工、能源、电网、自动化、基础设施、现代材料、日常工业。
- 不做枪炮武器链。
- 核能、输配电、机器人、PLC、传感器、质量检测属于本体现实近现代路线。
- 地球人是独立联动模组，不把真实人体难度塞进本体。
- 科幻附属在本体稳定后开发，承接太空、AI、纳米、反物质、曲率等超出现实工程的内容。

## 官方资料锚点

这些资料只作为玩法抽象的现实依据，不把现实危险、法规和复杂度原样搬进游戏：

- DOE Nuclear Energy: nuclear fuel cycle, used fuel and advanced reactor topics.
- NRC: commercial reactor types, PWR/BWR 基础、乏燃料池和干式贮存。
- EIA: 电力系统、发电类型、输配电基础。
- NIST: smart manufacturing, robotics, industrial control, measurement and quality inspection.
- DOE Industrial Decarbonization Roadmap: 钢铁、化工、炼油、水泥、食品饮料、纸浆造纸等高能耗工业部门。

参考入口：

- DOE Nuclear Energy fuel cycle: https://www.energy.gov/ne/nuclear-fuel-cycle
- EIA electricity explained: https://www.eia.gov/energyexplained/electricity/

## 核能玩法抽象

第一阶段不做危险模拟，只做“现实链条能看懂、能接进电网”的安全化路线：

1. 铀矿伴生入口：从花岗岩/独居石/尾粉/稀土尾渣中低概率路线得到铀矿物粉。
2. 浸出与黄饼：铀矿物粉 -> 浸出槽 -> 黄饼氧化铀 + 尾渣。
3. 转化与富集：黄饼 -> 反应釜 -> 六氟化铀；六氟化铀 -> 气体分离器 -> 低浓缩铀 + 贫铀。
4. 燃料制造：低浓缩铀 -> 反应釜/压粉机 -> 二氧化铀粉 -> 燃料芯块 -> 燃料棒 -> 燃料组件。
5. 核热与发电：燃料组件 + 控制棒/锆合金管/电路 -> 核热模块；核热模块可放入汽轮发电机，向 EOU 电网供电。
6. 乏燃料闭环：乏燃料组件 -> 冷却池/干式贮罐物品路线，默认安全封装，不主动伤害玩家。

玩家理解句：核电不是“铀锭烧一下”，而是把很长的燃料循环压缩成一条可玩的物品链，最后变成稳定大功率热源，再通过汽轮发电机进入电网。

## 自动化玩法抽象

自动化第一阶段服务现有机器，而不是另起复杂脚本系统：

1. 传感器：红石 + PCB + 玻璃/石英，表示温度、压力、料位、光学等传感入口。
2. PLC 控制器：PCB + 陶瓷基板 + 铜线，作为机器自动化核心件。
3. 执行器/伺服：铜线 + 钕铁硼磁材 + 钢/铝坯，表示电机、阀门和机械动作。
4. 机器人臂：伺服 + 铝合金坯 + PLC，用于后续自动上下料和多方块结构。
5. 视觉检测/质量模块：硅晶圆 + LED 荧光粉 + PCB，用于未来合格率、筛选、配方解锁或 JEI 说明。
6. 红石桥接：PLC 与红石比较器/侦测器互通，让玩家能用原版逻辑理解自动化。
7. 输送机：输送驱动机构 + 橡胶 + 钢坯 + 铜线 -> 工业输送机，第一版先推动掉落物和实体，后续再做分拣和接口升级。

玩家理解句：自动化不是要求玩家写代码，而是提供“看见、判断、动作、反馈”的现实工业部件。

## 本体开发优先级

1. 补齐现实定位文档、手册页、JEI 信息和 tooltip，让玩家先知道路线存在。
2. 增加核能和自动化中间物，接入现有机器，保证一条链能从已有材料跑到可见产物。
3. 增加变压器/汽轮机/蒸汽发生器等电网设备，形成核热/燃烧/电池并行的能源路线。当前已先落地汽轮发电机。
4. 增加输送机、分拣器、接口升级和更多多方块结构，使自动化从“材料”变成“功能”。当前已先落地基础输送机。
5. 做流体、多输入配比、催化剂耐久、热量和污染的可选软系统。
6. 主线足够稳定后，启动科幻附属。

## 科幻附属接入点

科幻附属不抢本体职责，而是读取本体产物：

- 高纯硅、晶圆、光刻胶、稀土、钕铁硼、PCB -> AI/机器人/计算设备。
- 核燃料、控制材料、储能、超导/导体 -> 高级能源。
- 钛、铝合金、陶瓷、碳材料、聚合物 -> 航天/结构材料。
- 自动化接口、传感器、PLC、机器人臂 -> 智能工厂、无人设施。

第一版科幻附属应该从“更高级用途”开始，不要反向修改本体开局难度。
